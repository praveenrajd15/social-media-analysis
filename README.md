
# 📊 Social Media Performance Analysis Dashboard

## 📌 Project Overview
This project analyzes social media performance data to evaluate engagement, reach, and audience interaction across platforms and regions.

## 🎯 Business Objectives
- Measure total engagement and reach
- Compare platform-wise performance
- Analyze region-based trends
- Identify high-performing content types
- Track monthly and quarterly growth

## 📂 Dataset Information
- Total Records: 1000
- Includes metrics like Likes, Comments, Shares, Followers, Reach, Engagement
- Contains platform, region, and time-based fields

## 🛠 Tools Used
- Power BI
- DAX
- Excel / CSV
- GitHub

## 📊 Key KPIs
- Total Likes
- Total Comments
- Total Shares
- Total Reach
- Total Engagement
- Platform-wise & Region-wise Analysis

## 🚀 Outcome
This dashboard helps marketing teams make data-driven decisions and optimize content strategy.
